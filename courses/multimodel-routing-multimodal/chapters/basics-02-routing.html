意图与级联

分类器选路由；级联先小后大。 关联：intent routing、cascade routing。

Fallback

主模型超时/429 切备用。 关联：fallback chain、router policy。

关键术语

intent routing · cascade routing · fallback chain · router policy · model selector · tiered llm · route rule · primary backup

示例

yaml
routing:
  rules:
    - intent: faq
      model: qwen-turbo
    - intent: vision_qa
      model: qwen-vl-plus
  fallback: [qwen-turbo, qwen-plus]
