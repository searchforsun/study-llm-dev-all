代码助手的最终用户界面是 IDE。如何将后端能力安全、低延迟地暴露给 VS Code / JetBrains 插件，涉及IDE 集成的插件边界（后端做什么 vs 前端做什么）、本地/云端部署策略、隐私保障（代码不出企业网络）。CorpAssist 的 IDE 插件覆盖 500+ 开发者，月均 10 万+ 次补全请求。

本章先记住 3 件事
  • 插件边界：IDE 插件只做 UI 展示和上下文提取，推理在后端。
  • 本地代理：在开发者机器上运行的轻量代理，负责缓存和质量检查。
  • 隐私第一：敏感代码走本地模型，日志中不存储完整代码。

VS Code 插件通信架构

CorpAssist · IDE 集成部署模型 企业内网部署：VS Code 插件 → 本地代理（每个开发者机器上运行） → 云端 API（企业内网部署）。插件提取上下文后发给本地代理，代理做缓存检查和质量过滤，再转发给云端 API。敏感文件（如支付核心代码）只走本地小模型，不出开发者机器。

# ide_bridge.py — CorpAssist IDE 本地代理
import hashlib
from fastapi import FastAPI, HTTPException
from pydantic import BaseModel, Field
from typing import Optional, List
import httpx
import ast

app = FastAPI(title="CorpAssist IDE 本地代理")

class LSPCompletionParams(BaseModel):
    uri: str = Field(..., description="file:// 格式的文件路径")
    position_line: int = Field(..., ge=0)
    position_character: int = Field(..., ge=0)
    context_before: str = Field(..., description="光标前 2000 字符")
    context_after: str = Field(default="", description="光标后 500 字符")
    language_id: str = Field(default="python")
    file_hash: Optional[str] = Field(None, description="文件内容的 SHA256 前缀(16位)")

class LSPCompletionItem(BaseModel):
    label: str
    text: str
    kind: int = 14  # Snippet
    detail: str = ""
    doc_string: str = ""

class IDEConnector:
    """IDE 连接器：本地代理逻辑"""

    SENSITIVE_PATTERNS = ["password", "secret", "token", "api_key", "private_key", "credential"]
    SENSITIVE_FILES = ["payment", "auth", "security", "crypto"]

    def __init__(self, cloud_api: str = "http://cloud-api:8000"):
        self.cloud_api = cloud_api
        self.cache = {}
        self.client = httpx.AsyncClient(timeout=10.0)

    def is_sensitive(self, uri: str, code: str) -> bool:
        """判断是否敏感文件"""
        uri_lower = uri.lower()
        for pattern in self.SENSITIVE_FILES:
            if pattern in uri_lower:
                return True
        for pattern in self.SENSITIVE_PATTERNS:
            if pattern in code.lower():
                return True
        return False

    def privacy_filter(self, code: str) -> str:
        """脱敏处理（替换敏感信息）"""
        import re
        code = re.sub(r'(password["\s:=]+)[^\s"\']+', r'\1***REDACTED***', code)
        code = re.sub(r'(api_key["\s:=]+)[^\s"\']+', r'\1***REDACTED***', code)
        code = re.sub(r'(secret["\s:=]+)[^\s"\']+', r'\1***REDACTED***', code)
        return code

    def check_syntax(self, code: str, language: str = "python") -> bool:
        """语法检查"""
        if language == "python":
            try:
                ast.parse(code)
                return True
            except SyntaxError:
                return False
        return True

    async def get_completion(self, params: LSPCompletionParams) -> List[LSPCompletionItem]:
        cache_key = hashlib.md5(f"{params.uri}:{params.position_line}:{params.context_before[:50]}".encode()).hexdigest()[:16]

        # 缓存命中
        if cache_key in self.cache:
            cached = self.cache[cache_key]
            return [LSPCompletionItem(label=cached[:30], text=cached, detail="(cache)")]

        # 隐私过滤 + 敏感检测
        code = params.context_before + params.context_after
        if self.is_sensitive(params.uri, code):
            code = self.privacy_filter(code)
            # 敏感文件走本地模型（此处模拟）
            completion = "# 敏感文件：本地模型处理\n# 实现逻辑略"
            if self.check_syntax(completion):
                self.cache[cache_key] = completion
                return [LSPCompletionItem(label=completion[:30], text=completion, detail="(local)")]

        # 普通文件走云端 API
        try:
            resp = await self.client.post(
                f"{self.cloud_api}/v1/completions",
                json={"prefix": params.context_before[-2000:], "suffix": params.context_after[:500],
                      "language": params.language_id, "max_tokens": 128, "temperature": 0.2},
                timeout=5.0,
            )
            data = resp.json()
            completion = data.get("completion", "")
            if self.check_syntax(completion):
                self.cache[cache_key] = completion
                return [LSPCompletionItem(label=completion[:50], text=completion, detail=params.uri)]
        except (httpx.TimeoutException, httpx.RequestError) as e:
            raise HTTPException(503, f"云端 API 不可用: {e}")

        return []

    async def close(self):
        await self.client.aclose()

connector = IDEConnector()

@app.post("/v1/completions", response_model=List[LSPCompletionItem])
async def complete(params: LSPCompletionParams):
    return await connector.get_completion(params)

@app.get("/health")
async def health():
    return {"status": "ok", "cache_size": len(connector.cache)}

if __name__ == "__main__":
    import uvicorn
    uvicorn.run(app, host="127.0.0.1", port=8100)
不推荐
  • IDE 插件直连云端 API（每次请求经过公网）
  • 所有代码明文发送到云端（隐私风险）
  • 不做本地缓存（每次请求都走推理）
  • 不检查语法，展示可能出错的补全
推荐
  • 本地代理层做缓存/隐私/质量检查
  • 敏感文件走本地模型，代码不出机器
  • 语义缓存 30% 命中，延迟 <5ms
  • 补全后做语法检查，不展示语法错误的代码

本章结论

  • 三层架构：IDE 插件（UI + 上下文提取）→ 本地代理（缓存 + 质检 + 隐私）→ 云端 API（推理）。
  • 插件边界：插件只负责展示和上下文提取，不做推理和决策。
  • 隐私保障：敏感文件（支付/认证/安全）走本地小模型，代码不出机器。
  • 质量网关：补全结果经过语法检查后再展示给开发者。
