Deployment

yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: corpassist-gateway
spec:
  replicas: 3
  template:
    spec:
      containers:
        - name: gateway
          image: registry/corpassist-gateway:1.2.0
          resources:
            requests: { cpu: "500m", memory: "512Mi" }
            limits: { cpu: "2", memory: "1Gi" }

HPA

除 CPU 外，可自定义 metrics：活跃 SSE 连接数、请求队列长度。app k8s 视角不深入 Prometheus Adapter 安装细节。

探针

  • liveness：/actuator/health/liveness — 进程死锁重启
  • readiness：依赖 Redis/Nacos 可达才接流量
  • 非 Infra 深度：不展开 CNI/Ingress Controller 调优

本章结论

  • Deployment：Gateway/BFF/Python 分 Deployment；资源 requests/limits 显式声明。
  • HPA：LLM 场景考虑连接数/custom metrics。
  • 探针：readiness 含依赖；滚动发布配合 SSE 重连。
