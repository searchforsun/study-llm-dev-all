运行手册

模型 API 全站 429 怎么办？
1) 确认厂商配额；2) 开 model fallback 小模型；3) 收紧 Gateway 限流；4) 公告 + 转人工。记录 incident。
SSE 大面积断开？
查 Gateway/BFF 滚动发布、Ingress 超时、Redis 会话；引导客户端 Last-Event-ID 重连。
