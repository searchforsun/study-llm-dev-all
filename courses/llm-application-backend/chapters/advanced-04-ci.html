为什么要学本章

Java Gateway 与 Python AI 须 dual stack ci 并行构建，contract gate 防止契约漂移。无 quality gate 的 LLM 服务发版风险极高。

学完你能
  • 绘制 cicd pipeline 阶段：build → test → image → deploy
  • 配置 release gate（评测通过率、契约测试）
  • 执行 blue green deploy 与 rollback ci

门禁

  • 单元/集成测试 green
  • 契约测试（汇合课 workshop）
  • 可选：Golden set eval pass rate ≥ 阈值（NS1）
  • 安全扫描：image build 无 critical CVE

本章结论

  • 双栈流水线：Java/Python 并行 build，汇合契约测试。
  • 门禁：测试 + 契约 + 可选 eval；失败阻断发布。
