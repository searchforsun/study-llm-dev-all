# Demo：spring-ai-alibaba-extensions

对应章节：[spring-ai-alibaba-extensions](../../index.html#ch-basics-03-extensions)

## 目标
集成 Document Parser 和 Vision 扩展，处理多模态输入。

## 操作步骤
1. 添加 `spring-ai-alibaba-document-parser` 依赖
2. 解析 PDF 文件输出 Markdown 结构化文本
3. 调用 `qwen-vl-plus` Vision 模型识别图片

## 验收
- Document Parser 成功解析 PDF
- Vision 模型正确描述图片内容
