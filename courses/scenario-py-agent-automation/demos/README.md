# CorpAssist Agent 自动化场景 · 实验目录

企业落地 Agent 任务自动化课程的配套动手实验。

| 实验 | 对应章节 | 核心练习 |
|------|----------|----------|
| practice-01-langgraph-lab | LangGraph 工作流实现 | 报销审批 StateGraph + 检查点恢复 |
| practice-02-saa-agent-lab | Spring AI Alibaba Agent 对照 | Java Python 双栈 Agent 对比实现 |
| practice-03-multi-tool-lab | 多工具协作与编排 | 会议安排并行工具编排 |
| practice-04-multi-agent-lab | 多 Agent 协作 | 采购审批 Supervisor + Specialist |
| practice-05-demo-lab | 完整 Agent 演示链路 | 流式 SSE 演示应用 |
| practice-06-production-pitfalls-lab | 生产可靠性 | 熔断器 + 降级策略验证 |

## 快速开始

```bash
cd demos/<实验目录>
pip install -r requirements.txt
python app.py
```
