模板引擎

prompt template engine 将人设、场景、变量占位解耦。CorpAssist 使用 {{company_name}}、{{tenant_id}} 等运行时注入。

yaml 
template_id: corpassist.greeting.v2
variables:
  company_name: {required: true}
body: |
  您好，我是 {{company_name}} 的 CorpAssist。
模板渲染流水线
flowchart LR
  Git[Git] --> Render[渲染]
  Render --> API[Chat API]

变量来自会话。

渲染前校验必填变量；缺省用 default。Python 用 Jinja2/LangChain；Java 用 StringTemplate 或 Spring 占位符。

  • 禁止在代码里拼接超长 System
  • 模板 ID 写入 trace 便于排障

回归基线

regression baseline：≥50 条固定用例，覆盖拒答、JSON、多轮指代。

json 
{"baseline_id":"corpassist-v1.4.2","cases":120,"thresholds":{"json_valid":0.98}}

发布闸门：baseline 失败禁止 Canary。变更后先 baseline 再 A/B（见 practice-04）。

无 baseline 上线？

无法归因 Prompt vs 模型，回滚只能全量。
