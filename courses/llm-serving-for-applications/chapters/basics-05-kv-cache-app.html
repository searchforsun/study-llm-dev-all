KV 直觉

每 token 存 K/V；越长占显存越多。 关联：kv cache、vllm kv param。

vLLM 参数

max-model-len、gpu-memory-utilization。 关联：latency vram、max seq len。

关键术语

kv cache · vllm kv param · latency vram · max seq len · app layer perf · prefix cache · no kernel dev · context memory

示例

yaml
vllm:
  max_model_len: 8192
  gpu_memory_utilization: 0.9
  enable_prefix_caching: true
