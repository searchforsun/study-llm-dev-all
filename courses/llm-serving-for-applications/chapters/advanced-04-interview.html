分层答法（2 分钟版）

  1. 契约：OpenAI /v1/chat/completions + FIM；双栈只改 base_url（Python、Spring）。
  2. 引擎：生产 vLLM + AWQ；引擎可比 TGI（advanced-01）。
  3. 发布：model alias + 网关灰度 + 指标回滚（practice-04）。
  4. 容量：并发粗算 + 显存 + GPU 数（advanced-03）。
  5. 混合：敏感代码走私有化，通用对话可 hybrid cloud fallback（混合 Serving）。

    参考答案

    缺 infer architecture 治理层：鉴权、超时、灰度、观测、双栈契约与 deploy pitfall 预案。
