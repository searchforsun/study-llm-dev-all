# 文档类型 → 处理路径决策表（请填写）

| 文档类型 | 页数/大小 | 是否反复问答 | 推荐路径 | 理由（一句话） |
|----------|-----------|--------------|----------|----------------|
| 扫描 PDF 制度 | 50 页 | 是 | OCR 入库 + RAG | 摊薄多次提问 Token |
| 报销单照片 | 1 张 | 否（单次字段） | Vision API | 页数少、字段抽取 |
| 临时合同 PDF | 8 页 | 否 | 异步 OCR → 会话 chunk | 未入库、会话级 |
| 含 L4 密级附件 | 任意 | — | 拒答 / 转人工 | 合规审批未通过 |

## 成本对比笔记

- OCR 入库：一次性成本，后续走文本 RAG（`corpassist-fast`）
- Vision 单次：按图像 patch 计 Token，适合 ≤2 页临时抽取
- 超过 10 页扫描件：必须引导正式入库或选页，避免撑爆上下文窗口
