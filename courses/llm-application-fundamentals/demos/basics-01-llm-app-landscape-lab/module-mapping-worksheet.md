## 参考答案要点

| 模块 | 面试场景 | 主要能力 | 是否依赖 S1 |
|------|----------|----------|-------------|
| S1 | 企业 RAG 知识库 | 索引、混合检索、引用溯源 | 自身即数据层 |
| S2 | 智能客服 | 意图路由、工单工具、人工转接 | 制度类问题常需 S1 |
| S3 | 办公自动化 | LangGraph / Agent、MCP 工具 | 可选检索制度 |
| S4 | 代码辅助 | 仓库索引、补全问答 | 独立代码索引 |
| S5 | 内容生成 | 模板、审核流水线 | 可用素材库 RAG |
