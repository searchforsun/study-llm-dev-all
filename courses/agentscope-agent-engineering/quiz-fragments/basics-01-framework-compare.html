1. （单选）AgentScope 2.0 AOP 编程模型的核心抽象是？

回顾「AOP 编程模型」Mermaid 图中心节点。

答案：B — AgentScope 以 Agent 为中心，Msg 传递消息，Event 驱动可观测性；StateGraph 是 LangGraph 心智。

2. （多选）CorpAssist S3 更适合优先 AgentScope 而非纯 LangGraph 的情况包括？

见「框架选型决策流」与决策表 CorpAssist 倾向列。

答案：A、C — 多 Agent 协作与 MCP/事件流是 AgentScope 强项；纯单轮 LLM 不必上 Agent 框架。

3. （填空）AgentScope 中表示用户输入的统一类型是________。

对照 LangGraph HumanMessage 的那一行表。

答案：UserMsg（接受 UserMsg 或 user msg 等变体）

见动手练习判断题与 FAQ 渐进迁移。
