1. （单选）AgentScope 单 session 多轮对话由哪个组件持有？

见对话历史管理小节。

答案：B — Memory 持有 UserMsg/AgentMsg 链。

2. （多选）Agentic 长期记忆适用于？

大 JSON 应截断，不应长期原样存储。

答案：A、C — 长期记忆存关键事实与偏好；B 应摘要或放日志。

3. （填空）Memory 配置________可在接近上下文上限时触发裁剪或摘要。

见裁剪策略配置代码块。

答案：max_tokens（或 summarize_after_tokens 等同类配置）

4. （单选·场景）周三用户说「按我的模板导出」，周一偏好存在长期 memory 中。这对应系统课的？

见衔接表与跨天场景 aside。

答案：A — 跨 session 偏好属于 Semantic/长期记忆。
