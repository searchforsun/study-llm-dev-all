ReactAgent 实现办公 Agent

Spring AI Alibaba 的 ReactAgent 封装了完整的 ReAct 循环。开发者只需注册 ToolCallback，Agent 自动处理思考-行动-观察循环。

java
// CorpAssistAgentConfiguration.java
@Configuration
public class CorpAssistAgentConfiguration {

    @Bean
    public ReactAgent corpAssistAgent(
            DashScopeChatModel chatModel,
            List<ToolCallback> tools) {
        return ReactAgent.builder()
            .chatModel(chatModel)
            .tools(tools)
            .maxSteps(12)
            .build();
    }

    @Bean
    public ToolCallback leaveBalanceTool() {
        return ToolCallback.builder()
            .name("query_leave_balance")
            .description("查询员工年假余额")
            .inputSchema("""
                {"type":"object","properties":{"employeeId":{"type":"string"}},
                 "required":["employeeId"]}""")
            .tool((ToolCallback.ToolAction<String, String>) input -> {
                QueryRequest req = JsonUtil.parse(input, QueryRequest.class);
                return leaveService.getBalance(req.getEmployeeId());
            })
            .build();
    }

    @Bean
    public ToolCallback sendEmailTool(ApprovalService approvalService) {
        return ToolCallback.builder()
            .name("send_email")
            .description("发送邮件（外部需审批）")
            .inputSchema("""
                {"type":"object","properties":{"to":{"type":"string"},
                 "subject":{"type":"string"},"body":{"type":"string"}},
                 "required":["to","subject"]}""")
            .tool((ToolCallback.ToolAction<String, String>) input -> {
                EmailRequest req = JsonUtil.parse(input, EmailRequest.class);
                if (isExternal(req.getTo())) {
                    return approvalService.createTask(req);
                }
                return emailService.send(req);
            })
            .build();
    }
}

Java：调用 Agent 服务

java
// AgentController.java
@RestController
@RequestMapping("/api/agent")
public class AgentController {

    private final ReactAgent agent;

    public AgentController(ReactAgent agent) {
        this.agent = agent;
    }

    @PostMapping("/execute")
    public String execute(@RequestBody AgentRequest request) {
        var messages = List.of(new UserMessage(request.getInput()));
        var prompt = new Prompt(messages);
        var response = agent.call(prompt);
        return response.getResult().getOutput().getContent();
    }
}

    java agent s3 中审批集成通过 Spring AOP 实现：

    java
    // ApprovalAspect.java — 审批切面
@Aspect
@Component
public class ApprovalAspect {

    private final ApprovalService approvalService;

    @Around("@annotation(RequireApproval)")
    public Object checkApproval(ProceedingJoinPoint pjp) throws Throwable {
        MethodSignature sig = (MethodSignature) pjp.getSignature();
        RequireApproval ann = sig.getMethod()
            .getAnnotation(RequireApproval.class);

        // 检查是否需要审批
        if (approvalService.needsApproval(ann.value(), pjp.getArgs())) {
            String taskId = approvalService.createTask(
                ann.value(), pjp.getArgs());
            // 挂起当前线程，等待审批
            ApprovalResult result = approvalService.waitForApproval(taskId);
            if (!result.isApproved()) {
                throw new ApprovalRejectedException(result.getReason());
            }
        }
        return pjp.proceed();
    }
}

    SAA 流式处理与 SSE 集成

    Spring AI Alibaba 支持流式（Streaming）方式返回 Agent 执行结果。这在办公场景中非常实用：用户可以实时看到 Agent 的思考过程和工具调用结果，而不是等待最终答案。

    java
    // StreamingAgentController.java
@RestController
@RequestMapping("/api/agent/stream")
public class StreamingAgentController {

    private final ReactAgent agent;

    @GetMapping(produces = MediaType.TEXT_EVENT_STREAM_VALUE)
    public Flux<ServerSentEvent<String>> streamExecute(
            @RequestParam String input) {
        var messages = List.of(new UserMessage(input));
        var prompt = new Prompt(messages);

        return agent.stream(prompt)
            .map(response -> {
                var output = response.getResult().getOutput();
                String content = output.getContent();

                // 判断是否需要工具调用
                if (output.hasToolCalls()) {
                    return ServerSentEvent.<String>builder()
                        .event("tool_call")
                        .data(output.getToolCalls().toString())
                        .build();
                }
                return ServerSentEvent.<String>builder()
                    .event("message")
                    .data(content)
                    .build();
            })
            .doOnError(e -> log.error("Stream error", e));
    }
}

    Python 对等的流式实现参考 完整 Agent 演示链路 中的 Streamlit 前端集成。SAA 的流式 API 基于 React 的 Flux 规范，前端可以用 EventSource 或 fetch + ReadableStream 消费。

    前端消费示例（JavaScript）：

    javascript
    const eventSource = new EventSource(
    `/api/agent/stream?input=${encodeURIComponent('请下周三的年假')}`
);
eventSource.addEventListener('message', (e) => {
    console.log('Agent 回复:', e.data);
});

    双栈部署的 Docker Compose 配置

    dual stack s3 的典型部署架构包含 Python LangGraph 服务 + Java SAA 服务 + 消息队列 + 数据库。以下是 Docker Compose 配置：

    yaml

version: '3.8'
services:
  agent-orchestrator:
    build:
      context: ./python-agent
      dockerfile: Dockerfile

      - "8000:8000"

    environment:

    Python 服务负责 Agent 编排和工具调用，Java 服务负责 HTTP 入口、审批管理和数据持久化。两者通过 RocketMQ 交换 AgentTask 和 WorkerResult 消息。

    本章结论

    • Spring AI Alibaba ReactAgent 封装标准 ReAct 循环，适合标准场景
    • DashScope 模型通过 application.yml 配置，支持 qwen-plus 等多模型
    • ToolCallback 自定义办公工具，审批集成通过 Spring AOP 注解
    • SAA 支持流式 SSE 输出，前端可实时展示 Agent 执行过程
    • 双栈部署通过 Docker Compose 编排，Python 编排 + Java 治理
    • LangGraph 适合复杂编排分支，SAA 适合标准 Java 企业治理

    常见问题（FAQ）

    SAA 的 ReactAgent 和 LangGraph 的 ReactAgent 有什么区别？
    SAA ReactAgent 是轻量封装，自动处理 tool call 循环，适合标准场景。LangGraph ReactAgent 可自定义状态和边，适合复杂分支。SAA 目前不支持 interrupt 原生，需要自己实现审批挂起。
    SAA 可以用非 DashScope 模型吗？
    可以。Spring AI Alibaba 支持通过 Spring AI 的 ChatModel 抽象接入多种模型（OpenAI、Ollama 等）。但 DashScope 集成最完善，包括 function calling 和 streaming 原生支持。
    Java 和 Python Agent 如何通信？
    推荐通过消息队列（RocketMQ/Kafka）或 MCP 桥接。Python 的复杂 Agent 处理结果通过事件发送给 Java 侧的审批服务，Java 的审批结果通过回调通知 Python。
