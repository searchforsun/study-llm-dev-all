检查点与恢复

每步 checkpoint；失败从上次恢复。 关联：checkpoint、failure recovery。

关键术语

示例

python
plan = ["查假期余额", "填写请假单", "提交审批"]
for step in plan:

    if result.failed:
        break  # 或 replan
