# Demo：办公自动化场景与边界

[章节](../../index.html#ch-basics-01-business)

## 目标

完成办公场景自动化边界的分类练习，掌握 task boundary 的四象限划分方法。

## 练习内容

1. **边界填表**（boundary-table.md）：将 10 个办公操作填入四象限矩阵，标注「全自动」「草案+审批」「人工」
2. **审批规则 YAML**：配置 3 条审批策略（对外邮件、请假超 3 天、付款超 5000）
3. **代码演练**：用 Python 或 Java 实现审批规则评估函数

## 验收标准

- 四象限表至少包含 8 个正确分类的操作
- 审批 YAML 符合 CorpAssist 格式，能被 Python `approval_config.py` 解析
- 代码演练：评估函数能正确判断是否需要审批

## 场景示例

| 操作 | 分类 | 理由 |
|------|------|------|
| 查请假余额 | 全自动 | 只读、低风险 |
| 发对外邮件 | 草案+审批 | 对外沟通、不可逆 |
| 批量改权限 | 人工 | 高风险、影响面大 |
