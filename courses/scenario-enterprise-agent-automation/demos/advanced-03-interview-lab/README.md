# Demo：面试：Agent 五连问

[章节](../../index.html#ch-advanced-03-interview)

## 目标

通过五连问模拟面试，系统准备 Agent 架构师岗位的面试。

## 练习内容

1. **回答提纲**（interview-outline.md）：为 Agent 五连问（区别、架构、规划、记忆、工具失败）各写一份完整回答提纲
2. **模拟面试**：录制 5 分钟视频，完整回答五连问，自评流畅度和深度
3. **STAR 故事打磨**：将你的 Agent 项目用 STAR 框架写成 3 分钟可以讲完的 delivery story

## 验收标准

- 每个回答 2-3 分钟，先给结论再展开
- 记忆题必须提到三层结构（短期/近期/长期）的容量参数
- 工具失败题必须包含熔断→重试→降级→人工→审计五步

## 五连问框架

| 问题 | 核心回答 | 时间 |
|------|---------|------|
| Agent vs 对话 | 有状态+调工具+循环推理+可恢复 | 2 分钟 |
| 架构设计 | 选型理由+组件图+数据流+可靠性 | 3 分钟 |
| 任务规划 | Planner 分解+检查点+动态调整 | 2 分钟 |
| 记忆系统 | 三层结构+窗口+摘要+向量 | 2 分钟 |
| 工具失败 | 熔断→重试→降级→人工→审计 | 3 分钟 |
