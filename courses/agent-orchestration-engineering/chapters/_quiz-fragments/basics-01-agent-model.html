1. （单选）S2 用户问「8821 退款到哪了」，CRM 已返回 pending，但模型在下一轮仍说「查不到」。最可能缺了哪一环？

ReAct 里 Observation 必须被下一轮「感知」到。

答案：B — 未回写 observation 时 LLM 看不见工具结果，会幻觉。

2. （多选）CorpAssist 要在 Python 轨落地 S3 多步审批 + MCP，同时 Java 网关保留简单 FAQ。合理做法包括？

对照本章「双栈」表：能力对齐、契约对齐。

答案：A、C — B 分裂契约；D 不利排障。

3. （填空）在感知-规划-行动模型中，LLM 根据 messages 决定调用 query_refund_status 属于________阶段（填中文二字）。

对应 ReAct 的 Thought/plan，不是 tool 执行本身。

答案：规划（接受：规划）

4. （单选）同一 S2 会话第 6 次仍重复调用 query_refund_status 且参数相同。生产上应先加？

本章「反例 vs 推荐」强调步数预算。

答案：C — 防止 react loop 失控与成本飙升。
