1. （单选）CorpAssist S2 促销高峰，优先应控制哪两类预算？

步数防 react 死循环，Token 防长上下文；缺任一都可能成本失控。

答案：B — Agent 成本主要来自 LLM 调用次数与 Token，须双预算。

2. （单选）S2「查物流」适合哪种模型路由？

cheap intent 在图入口一次路由，简单 FAQ 走小模型。

答案：A — 高置信简单意图用 turbo，步数 cap 4～6。

3. （多选）步数耗尽时的合理生产行为？

超限应有用户可感知降级，而非异常崩溃。

答案：A、C — 转人工 + 图内 degrade 是 S2 标准做法。

4. （填空）验证模型路由降本效果，应在 LangSmith/Prometheus 部署________记录 model 与 intent 维度。

课程术语：按模型与意图计量用量。

答案：usage meter（usage meter / 用量计量）
