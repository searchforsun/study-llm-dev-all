resume

python
# 审批通过后
graph.invoke(
    Command(resume={"approved": True, "approver": "ops-12"}),
    config={"configurable": {"thread_id": thread_id}},
)

拒绝时 resume approved: false，路由到解释节点或 handoff。

审计

每次 human approval 记录：thread_id、tool_name、args 摘要、approver、时间戳。写入 pending state 表供合规查询。

字段用途
trace_id与 LangSmith/Micrometer 关联
payload_hash防篡改对比
decisionapproved / rejected / timeout

本章结论

  • interrupt：写操作前强制暂停。
  • resume：审批结果驱动继续或终止。
  • 审计：全链路可追溯，满足合规。
