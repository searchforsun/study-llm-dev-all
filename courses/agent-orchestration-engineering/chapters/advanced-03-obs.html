为什么要学本章

Agent 排障离不开逐步 trace。LangSmith 承担 Python 轨 observability，支撑 NS1 坏例飞轮。

学完你能

trace

环境变量 LANGCHAIN_TRACING_V2=true；每个 graph invoke 一条 run，节点为 child span。

bash
export LANGCHAIN_TRACING_V2=true
export LANGCHAIN_API_KEY=...
export LANGCHAIN_PROJECT=corpassist-s2-agent

数据集

Golden 对话 + 期望工具调用序列；feedback loop 把生产 badcase 追加进 dataset。

对比

发版前跑 eval：对比 tool 选择准确率、步数分布。Trace replay 复现用户会话。

本章结论

  • trace：逐步可见，排障基础。
  • 数据集：坏例进集，NS1 飞轮。
  • 对比：发版门禁量化。
