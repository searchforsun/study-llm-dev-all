主管-工人

Orchestrator（supervisor）只持有路由与汇总工具；worker agent 持有窄工具集。

Supervisor

Delegation、步数预算、最终答复。

Research Worker

RAG/检索，只读工具。

Action Worker

写操作 + interrupt 审批。

Handoff

LangGraph 可用 Command 或子图 invoke 实现 handoff：supervisor 在 state 写 next_agent，条件边进入 worker 子图。

python
def supervisor(state):
    decision = llm.invoke(state["messages"])
    if decision.worker == "research":
        return Command(goto="research_worker")
    return Command(goto="action_worker")

Role split 原则：每 worker ≤5 个工具；写操作集中在 action worker。

先想 10 秒：多 Agent 是否一定比单 Agent 好？

否。工具 <8 且意图清晰时单 Agent 更简单；多 Agent 增加 trace 与延迟，用于 S3 长流程。

本章结论

  • 主管-工人：supervisor 委派，worker 窄工具。
  • Handoff：state/Command 驱动子图切换。
  • SAA 对照：语义对齐，契约与 trace 贯通双栈。
