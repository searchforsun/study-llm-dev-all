# 多 Agent 协作模式实验室

## 目标
实现 Routing + Sequential 混合模式的多 Agent 系统。

## 练习
1. 实现路由器 Agent（意图识别与分发）
2. 实现三个专业 Agent（客服/售后/技术支持）
3. 实现 Sequential 退款处理流水线
4. 实现 Supervisor-Worker 汇报生成

## 验收标准
- [ ] 路由器正确识别意图并分发
- [ ] 专业 Agent 处理各自领域
- [ ] Handoff 时上下文完整传递
- [ ] 测试混合模式下的协作流程
