项目经理

RACI 与 solution doc 版本对齐 discovery 纪要。

不推荐
  • Python 与 Spring 各维护一套 Prompt
  • BFF 直接调 DashScope 绕过 RAG
推荐
  • 契约规定 /v1/chat 仅 BFF→Python
  • Prompt 变更走 PR + golden 回归

接口

openapi contract 是双栈主 integration point。PoC 前建议冻结 /v1/chat、/v1/ingest/status、/v1/eval/run。

yaml
openapi: 3.0.3
paths:
  /v1/chat:
    post:
      summary: CorpAssist 问答（BFF 调用）
      parameters:
        - in: header
          name: X-Trace-Id
          required: true
      requestBody:
        content:
          application/json:
            schema:
              properties:
                query: { type: string }
                deptId: { type: string }
                citationsRequired: { type: boolean, default: true }
  • 版本：URL 前缀 /v1；破坏性变更升 v2。
  • 错误码：4xx 可重试策略；5xx 触发 BFF 降级话术。
  • 门禁：consumer/provider 契约测试进 CI。
客户要求「接口文档放二期」？
PoC 可简化为 3 路径 Markdown 附件，但字段名与验收指标不得口头漂移，否则 验收与 SLA 无法对齐。

本章结论

三条对应上文：架构图、分工、接口。

  • 架构图：L1 业务视图 + L2 部署；标注 SSO、向量库、通义与审计链。
  • 分工：Python 管道与评测；Spring 编排、鉴权、对外 SLA。
  • 接口：OpenAPI 冻结路径；契约测试先于大规模联调。
