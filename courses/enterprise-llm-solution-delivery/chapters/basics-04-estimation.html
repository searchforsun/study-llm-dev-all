Token

token cost = 月请求量 ×（输入+输出 Token/次）× 单价。CorpAssist 制度问答可按 2k 输入 + 500 输出估算。

python
# 月 Token 成本粗算（示例）
monthly_queries = 50_000
tokens_per_query = 2500  # 含检索上下文
price_per_1k = 0.008  # 按厂商报价表
monthly_cost = monthly_queries * tokens_per_query / 1000 * price_per_1k
不推荐
  • 只估对话 Token 忽略检索注入
  • 按峰值 QPS 当平均
推荐
  • 分场景（问答 vs 批处理）
  • BFF 记录 usage 回传对账

硬件

hardware budget 与 private deploy 绑定；公有云以 infra cost（向量库、对象存储、计算）为主。
