# Demo：持续学习路线图

[返回章节](../index.html#ch-advanced-05-roadmap)

## 目标

制定个人 90 天学习路线图，明确算法/Infra 边界与社区资源。

## 验收标准

- [ ] `my-roadmap.md` 含 3 个月标题
- [ ] 每月 ≥3 条 checkbox
- [ ] 至少 1 处 NS 或 S 标签
- [ ] 选定主分支（Delivery / AppEng / Algo / Infra）

## 产出物

| 文件 | 说明 |
|------|------|
| `my-roadmap.md` | 个人 90 天路线图 |

## 操作步骤

1. 选定主分支
2. 列出最弱 NS 与最浅 S
3. 抄录 90 天模板到 my-roadmap.md
4. 从社区表选 3 条资源
5. 完成章节测验
