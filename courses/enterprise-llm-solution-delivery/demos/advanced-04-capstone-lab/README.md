# Demo：毕业答辩：方案评审

[返回章节](../index.html#ch-advanced-04-capstone)

## 目标

用 rubric 自评方案并完成 peer defect 草稿。

## 验收标准

- [ ] `rubric-self-score.md` 含四维分数 + 总分
- [ ] 每维写 1 句理由
- [ ] 另附 peer defect 3 条或 incorporate 行动 3 条
- [ ] 至少 1 条 P0/P1 defect

## 产出物

| 文件 | 说明 |
|------|------|
| `rubric-self-score.md` | 四维自评表 |
| `peer-defects.yaml` | Peer defect 清单 |

## 操作步骤

1. 为自有方案四维打分
2. 对照三座大山速查表自检
3. 写 3 条 scheme defect
4. 写 24h incorporate 行动
