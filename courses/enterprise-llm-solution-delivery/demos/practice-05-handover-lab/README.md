# Demo：移交与运维

[返回章节](../index.html#ch-practice-05-handover)

## 目标

为 CorpAssist 创建移交文档清单、Runbook 与版本升级流程。

## 验收标准

- [ ] `handover-checklist.md` 含 6 模块文件列表
- [ ] 至少 1 条完整 Runbook（症状→排查→恢复→验证）
- [ ] 含版本升级检查清单（≥5 步）
- [ ] 含 Incident Flow 描述

## 产出物

| 文件 | 说明 |
|------|------|
| `handover-checklist.md` | 移交文档 6 模块清单 |
| `runbook-example.md` | 典型故障 Runbook（如模型超时） |
| `upgrade-checklist.md` | 版本升级检查清单 |
| `incident-flow.md` | 故障响应流程 |

## 操作步骤

1. 创建 6 模块移交文件列表
2. 写一条 BFF 不可用的 Runbook
3. 设计版本升级检查清单
4. 画出 Incident Flow 文字描述
