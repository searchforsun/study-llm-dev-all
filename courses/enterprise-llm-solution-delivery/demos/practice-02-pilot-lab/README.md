# Demo：试点上线

[返回章节](../index.html#ch-practice-02-pilot)

## 目标

为 CorpAssist 设计灰度发布计划、用户培训安排与值班制度。

## 验收标准

- [ ] `pilot-plan.md` 含三阶段灰度计划（比例/时间/门禁）
- [ ] 含 Champion User 培训议程
- [ ] 含 L1/L2/L3 值班表
- [ ] 含 3 个 Pilot KPI 目标值

## 产出物

| 文件 | 说明 |
|------|------|
| `pilot-plan.md` | 灰度发布 + 回滚条件 |
| `training-agenda.md` | 培训计划（三层） |
| `oncall-schedule.md` | 值班表 + 升级路径 |
| `pilot-kpi.md` | 试点 KPI 看板 |

## 操作步骤

1. 设计三阶段灰度计划
2. 写 Champion User 培训议程
3. 设计值班表（4 人团队 + 客户 IT 1 人）
4. 定义 3 个 Pilot KPI 目标值
