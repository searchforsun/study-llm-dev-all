# Demo：PoC 设计与演示

[返回章节](../index.html#ch-practice-01-poc)

## 目标

为 CorpAssist 编写 PoC 演示脚本、指标看板模板与反馈问卷。

## 验收标准

- [ ] `poc-script.md` 含 3 场景脚本 + 指标目标值
- [ ] 每个场景含客户关注点与应对话术
- [ ] 至少 1 个异常场景的 Plan B
- [ ] 会后 48h 反馈邮件模板

## 产出物

| 文件 | 说明 |
|------|------|
| `poc-script.md` | 3 场景 × 5 分钟演示脚本 |
| `demo-metrics.md` | PoC 指标看板目标值 |
| `feedback-survey.md` | 48h 反馈问卷模板 |

## 操作步骤

1. 为 S1/S2/ACL 三个场景编写脚本
2. 设定 PoC 指标目标值
3. 准备 1 个异常场景的 Plan B
4. 写会后跟进邮件模板
