# Demo：交付反模式

[返回章节](../index.html#ch-advanced-03-anti-patterns)

## 目标

识别 CorpAssist 交付中的过度承诺和技术债，填写反模式清单。

## 验收标准

- [ ] `anti-pattern-list.md` ≥5 行
- [ ] 每行含反模式名、后果、缓解措施
- [ ] 含 over promise 与 tech debt 各至少 1 条
- [ ] 含 tech_debt.yaml 格式的 2 条债项

## 产出物

| 文件 | 说明 |
|------|------|
| `anti-pattern-list.md` | 反模式清单 + 缓解措施 |
| `tech-debt-example.yaml` | 技术债登记示例 |

## 操作步骤

1. 从过度承诺表选 2 行，写缓解措施
2. 从技术债表选 3 个债项，填 yaml
3. 勾选试点前反模式扫描清单
4. 标记未关闭项的 owner
