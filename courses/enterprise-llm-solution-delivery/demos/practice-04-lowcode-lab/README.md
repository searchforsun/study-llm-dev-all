# Demo：Dify/Coze 与自研取舍

[返回章节](../index.html#ch-practice-04-lowcode)

## 目标

为 CorpAssist 做出低代码 vs 自研的决策，并输出混合架构方案。

## 验收标准

- [ ] `build-vs-buy.md` 含 4 维决策矩阵
- [ ] 含混合架构图描述
- [ ] 含推荐结论
- [ ] 至少 2 项选 Buy、2 项选 Build

## 产出物

| 文件 | 说明 |
|------|------|
| `build-vs-buy.md` | 4 维决策矩阵 + 推荐结论 |
| `hybrid-arch.md` | 混合架构文字描述 |
| `escape-hatch.yaml` | 逃生门配置（可选） |

## 操作步骤

1. 用四维矩阵做决策
2. 画出混合架构图
3. 写推荐结论描述
4. （可选）设计 escape hatch 配置
