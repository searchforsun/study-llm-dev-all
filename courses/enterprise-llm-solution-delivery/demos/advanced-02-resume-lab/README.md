# Demo：简历与面试叙事

[返回章节](../index.html#ch-advanced-02-resume)

## 目标

基于 CorpAssist 案例编写双轨 bullet 与 STAR 故事。

## 验收标准

- [ ] `star-draft.md` 含 S/T/A/R 四标题
- [ ] R 段有数字指标
- [ ] 另附交付轨、技术轨 bullet 各 1 条
- [ ] 至少 2 个模拟追问应答

## 产出物

| 文件 | 说明 |
|------|------|
| `star-draft.md` | 完整 STAR 四段 + double bullet |
| `interview-qa.md` | 模拟追问应答 ×2 |

## 操作步骤

1. 写交付轨 bullet（含阶段+数字）
2. 写技术轨 bullet（含双栈+技术决策）
3. 填 STAR 四段
4. 写 2 个追问应答
5. 90 秒口述计时
