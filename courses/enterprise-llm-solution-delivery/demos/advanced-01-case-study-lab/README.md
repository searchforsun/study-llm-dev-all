# Demo：CorpAssist 完整案例

[返回章节](../index.html#ch-advanced-01-case-study)

## 目标

将 CorpAssist 的完整交付故事写入 `case-outline.md`，包含双栈分工、三阶段指标与复盘。

## 验收标准

- [ ] `case-outline.md` 含背景、双栈、指标、复盘四段
- [ ] 双栈段提及 OpenAPI 或 ACL
- [ ] 指标段区分 PoC/试点/合同三类
- [ ] 复盘段含 1 条决策+影响+行动

## 产出物

| 文件 | 说明 |
|------|------|
| `case-outline.md` | CorpAssist 完整案例大纲 |

## 操作步骤

1. 在双栈分工表中补充职责
2. 从三阶段指标表中各抄 1 个指标
3. 写 1 条「决策+影响+行动」复盘
4. 可选：导出 PDF 供互评
