政策库 RAG：Spring AI 检索增强

客服场景的政策问答与 S1 知识库共享同一套检索基础设施，但需要额外的「版本锁」确保回答基于最新政策。

java
@Service
public class PolicyRagService {

    private final VectorStore vectorStore;
    private final ChatClient chatClient;

    public Answer answerWithPolicy(String query, String department) {
        // 1. 检索最新政策（按版本过滤）
        SearchRequest request = SearchRequest.builder()
            .query(query)
            .topK(3)
            .filterExpression(FilterExpression.builder()
                .eq("department", department)
                .eq("status", "active")     // 只查生效政策
                .eq("policy_type", "customer_service")
                .build())
            .build();

        List policies = vectorStore.search(request);

        // 2. 如果没有相关政策，明确告知
        if (policies.isEmpty()) {
            return new Answer("抱歉，当前知识库中没有找到相关的政策信息。请转接人工坐席获取帮助。",
                List.of(), false);
        }

        // 3. 回答并强制引用政策原文
        return chatClient.prompt()
            .system("你是一个客服政策助手。请基于提供的政策文档回答。" +
                "每次引述政策时必须标注政策编号和生效版本。")
            .user(query)
            .call()
            .entity(Answer.class);
    }
}

话术模板与 MessageTemplate

java
@Component
public class ScriptTemplateManager {

    private final Map templates = new HashMap<>();

    @PostConstruct
    public void init() {
        // 退货话术 - 必须与政策严格一致
        templates.put("return_policy", MessageTemplate.builder()
            .name("return_policy")
            .template("""
                根据《退换货政策》(v{version})：
                1. 商品签收后 {returnDays} 天内可申请无理由退货
                2. 退货商品需保持原包装完好
                3. 运费承担：{shippingRule}

                请问您需要我为您创建退货申请吗？
                """)
            .build());

        // 道歉话术
        templates.put("apology", MessageTemplate.builder()
            .name("apology")
            .template("很抱歉给您带来不便。{reason}。我们会尽快为您处理。")
            .build());
    }

    public String render(String templateName, Map params) {
        MessageTemplate template = templates.get(templateName);
        if (template == null) {
            throw new IllegalArgumentException("Template not found: " + templateName);
        }
        return template.render(params);
    }
}

输出内容安全审核（Advisor 链）

java
@Component
public class ComplianceAdvisor implements Advisor {

    // 禁止承诺词列表
    private static final List FORBIDDEN_PROMISES = List.of(
        "一定退款", "保证赔偿", "100% 解决", "24小时内解决",
        "全额赔付", "免费升级", "无条件"
    );

    // 需要转人工的关键词
    private static final List SENSITIVE_KEYWORDS = List.of(
        "投诉", "律师", "起诉", "媒体", "曝光", "315", "工商"
    );

    @Override
    public AdvisedResponse aroundCall(AdvisedRequest request) {
        AdvisedResponse response = request.next().aroundCall(request);

        String content = response.getResponse();
        List violations = new ArrayList<>();

        // 检查禁止承诺
        for (String word : FORBIDDEN_PROMISES) {
            if (content.contains(word)) {
                violations.add("包含禁止承诺词: " + word);
            }
        }

        // 检查与政策的冲突
        if (hasPolicyConflict(content)) {
            violations.add("回答可能与最新政策冲突，请人工复核");
        }

        if (!violations.isEmpty()) {
            response.getAttributes().put("compliance_violations", violations);
            response.getAttributes().put("needs_review", true);
        }

        return response;
    }
}

Q1: 政策版本切换时正在进行的对话怎么处理？

不中断当前对话——已经开始的会话继续使用旧版本回答。新开会话使用新版本。政策版本切换后，旧索引保留 72 小时后删除。系统监听政策变更事件，生成通知提示坐席组。
