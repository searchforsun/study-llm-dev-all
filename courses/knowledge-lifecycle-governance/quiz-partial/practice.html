1. （单选）CorpAssist 日常 Confluence 正文小改，应优先采用哪种更新方式？

正文小改可用 content_hash 识别；Embedding 模型未变。

答案：B — 日常变更走增量；全量用于模型/schema 变更或对账修复。

2. （多选）下列哪些情况必须 full rebuild（而非仅 incremental）？

B/D 可用 incremental：upsert 或 delete by doc_id。

答案：A、C — 模型与分块策略变更需新 index 全量 re-embed。

3. （填空）Python 索引完成后通知 Spring 刷新 alias 的事件名是________（大写英文下划线）。

见正文 JSON 事件契约 event 字段。

答案：INDEX_PUBLISHED

4. （判断/场景）对账 diff 为 0.8%，gate 配置 max_diff_ratio=0.005。alias switch 应如何处理？

答案：B — 0.8% > 0.5% 阈值，须 block 切换。
