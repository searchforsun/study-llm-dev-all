为什么学这章

企业内容创作需要素材支撑——产品图片、往期文章、竞品资料、品牌规范。多模态素材 RAG 将这些非结构化素材统一管理，在生成内容时自动检索相关素材作为参考。

学完后能做什么

实现图文素材的统一索引和检索；配置版权合规检查（图片引用/文字引用）；调用多模态 API（文生图/图生文）；实现素材使用的版权追溯。

本章要点速记

1) 素材索引：图片 Clip Embedding + 文字 Embedding 统一存储；2) 版权合规：图片来源标记（自摄/素材库/AI生成），使用前检查权限；3) 多模态 API：DashScope 的文生图（qwen-vl-max）和图生文（qwen-vl-plus）；4) 版权追溯：每次素材使用记录素材 ID + 内容 ID + 使用时间。

图文素材库

// 素材索引
@Component
public class AssetIndexer {

    private final VectorStore vectorStore;
    private final ImageEmbeddingService imageEmbedding;

    public void indexImage(Asset asset) {
        // 图片 Embedding（Clip 模型）
        float[] imageVector = imageEmbedding.embed(asset.getFile());

        Document doc = new Document(
            asset.getDescription(),
            Map.of(
                "assetId", asset.getId(),
                "type", "image",
                "url", asset.getUrl(),
                "source", asset.getSource(),   // ORIGINAL / AI_GENERATED / STOCK
                "license", asset.getLicense(),
                "tags", String.join(",", asset.getTags()),
                "createdAt", asset.getCreatedAt().toString()
            )
        );
        doc.setEmbedding(imageVector);
        vectorStore.add(List.of(doc));
    }

    public void indexText(Asset asset) {
        Document doc = new Document(
            asset.getContent(),
            Map.of(
                "assetId", asset.getId(),
                "type", "text",
                "source", asset.getSource(),
                "url", asset.getUrl(),
                "tags", String.join(",", asset.getTags())
            )
        );
        vectorStore.add(List.of(doc));
    }

    public List<Asset> searchAssets(String query, String type, int topK) {
        SearchRequest request = SearchRequest.builder()
            .query(query)
            .topK(topK)
            .filterExpression("type == '" + type + "'")
            .build();
        return vectorStore.similaritySearch(request).stream()
            .map(this::toAsset)
            .toList();
    }
}

版权合规与多模态 API

// 版权检查器
@Component
public class CopyrightChecker {

    public CheckResult checkUsage(Asset asset, String contentId) {
        // 检查素材来源权限
        switch (asset.getSource()) {
            case "STOCK":
                // 素材库图片：检查授权是否过期
                if (asset.getLicenseExpiry().isBefore(Instant.now())) {
                    return CheckResult.deny("素材库授权已过期: " + asset.getId());
                }
                break;
            case "AI_GENERATED":
                // AI 生成图片：可自由使用
                break;
            case "ORIGINAL":
                // 自摄图片：检查是否有原始授权
                if (!asset.hasPermission()) {
                    return CheckResult.deny("自摄图片缺少使用授权");
                }
                break;
        }
        return CheckResult.allow();
    }

    // 记录使用
    public void recordUsage(Asset asset, String contentId, String userId) {
        UsageRecord record = new UsageRecord(
            UUID.randomUUID().toString(),
            asset.getId(), contentId, userId, Instant.now()
        );
        usageRepository.save(record);
    }
}

// 多模态 API 调用
@Component
public class MultiModalService {

    private final ChatClient vlClient;

    // 文生图：生成配图
    public String generateImage(String prompt, String style) {
        return vlClient.prompt()
            .user(u -> u.text("""
                生成一张配图。要求：
                描述: {prompt}
                风格: {style}
                比例: 16:9
                输出: 返回图片 Markdown 格式
                """)
                .param("prompt", prompt)
                .param("style", style)
            )
            .call()
            .content();
    }

    // 图生文：描述图片内容
    public String describeImage(String imageUrl) {
        return vlClient.prompt()
            .user(u -> u.text("请详细描述这张图片的内容、风格和适用场景")
                .media(Media.of(imageUrl, MediaType.IMAGE))
            )
            .call()
            .content();
    }
}

CorpAssist 场景

生成一篇产品发布文章时，素材 RAG 自动检索到：1) 产品实拍图（自摄，有授权）→ 插入配图；2) 往期发布文案 3 篇（参考风格）→ 注入 Few-shot；3) 品牌规范文档（色调/字体）→ 注入 System Prompt。如果有图片来自素材库且授权即将过期（7 天内），自动标记并在文章底部添加"本图片使用 Shutterstock 授权"的版权声明。整个素材检索+版权检查+图文生成流程约 5 秒。
