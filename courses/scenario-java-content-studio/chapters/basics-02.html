已完成风格控制与模板库

约 30 分钟 · 阶段：业务与架构 · 场景：S5 内容生成工坊

为什么学这章

企业内容需要统一品牌语调。直接让 LLM 生成的内容往往风格不一致。本章通过品牌语调参数、Few-shot 示例、Prompt 模板版本管理，实现可复用的风格控制体系。

学完后能做什么

定义品牌语调参数（正式度/情感度/专业度等）；用 Few-shot 示例控制生成风格；在 Git 中版本管理 Prompt 模板；使用 CorpAssist 营销模板库快速生成风格一致的内容。

本章要点速记

1) 品牌语调五维参数：正式度(0-10)、情感度(0-10)、专业度(0-10)、简洁度(0-10)、幽默度(0-10)；2) Few-shot 示例提供 2-3 个"好的"例子比描述要求更有效；3) Prompt 模板用 Git 管理版本，每次修改有 diff 记录；4) CorpAssist 预设 20+ 行业营销模板（电商/金融/教育/SaaS）。

品牌语调参数

java
// 品牌语调参数配置
@Data
@Builder
public class BrandTone {
    @Min(0) @Max(10) private int formality;      // 正式度 0=口语，10=官方
    @Min(0) @Max(10) private int emotion;         // 情感度 0=中性，10=热情
    @Min(0) @Max(10) private int professionalism; // 专业度 0=通用，10=专业
    @Min(0) @Max(10) private int conciseness;     // 简洁度 0=详细，10=极简
    @Min(0) @Max(10) private int humor;           // 幽默度 0=严肃，10=诙谐

    public String toPromptInstruction() {
        return String.format("""
            写作风格要求：
            - 正式度: %d/10（%s）
            - 情感度: %d/10（%s）
            - 专业度: %d/10（%s）
            - 简洁度: %d/10（%s）
            - 幽默度: %d/10（%s）
            """,
            formality, describe(formality, "口语化", "中性", "官方正式"),
            emotion, describe(emotion, "冷静", "温和", "热情"),
            professionalism, describe(professionalism, "通用", "行业", "深度专业"),
            conciseness, describe(conciseness, "详细", "适中", "极简"),
            humor, describe(humor, "严肃", "轻松", "诙谐"));
    }

    private String describe(int value, String low, String mid, String high) {
        if (value <= 3) return low;
        if (value <= 7) return mid;
        return high;
    }

    // 预设语调
    public static BrandTone CORPORATE_OFFICIAL = BrandTone.builder()
        .formality(9).emotion(4).professionalism(8)
        .conciseness(5).humor(1).build();

    public static BrandTone SOCIAL_MEDIA = BrandTone.builder()
        .formality(3).emotion(8).professionalism(4)
        .conciseness(7).humor(6).build();
}

Few-shot 示例与模板版本管理

java
// Few-shot 示例模板
@Component
public class FewShotTemplate {

    public String buildPrompt(String scenario, BrandTone tone,
                               Map<String, String> variables) {
        return String.format("""
            请根据以下要求生成内容。

            【品牌语调】
            %s

            【示例 - 好的范例】
            示例1（产品发布）：
            "我们很高兴地宣布 CorpAssist 3.0 正式发布。
            本次更新带来了三大核心能力：多 Agent 协作、
            审批节点集成、MCP 工具生态。"
            特点：简洁、信息明确、用"我们"体现团队感

            示例2（活动通知）：
            "邀请你参加 5 月 20 日的 Agent 自动化实践分享会，
            我们将现场演示如何用 Spring AI Alibaba 构建 Agent 系统。"
            特点：行动导向、时间明确、突出价值

            【当前任务】
            场景: %s
            关键信息: %s

            请严格按照示例的风格和结构生成。
            """,
            tone.toPromptInstruction(),
            scenario,
            formatVariables(variables));
    }
}
