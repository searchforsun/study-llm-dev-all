# Demo：面试——召回低排查实验

**章节**：[../../index.html#ch-advanced-05-interview]面试：召回低怎么排查

## 目标

用排查 checklist 解决模拟的召回问题，用 STAR 框架写 30 秒面试答法。

## 实验内容

1. **排查框架练习**：给定一个「年假政策搜索不到」的模拟问题，按八步排查清单逐层检查。
2. **Embedding Mismatch 模拟**：故意用 text-embedding-ada-002（而非 3-small）查询，观察召回变化。
3. **Chunk 问题模拟**：设 chunk_size=128，观察关键短语被切断后的召回影响。
4. **Checklist 输出**：写一份排查 checklist，覆盖至少 6 个检查项，每项有验证方法。
5. **面试答法撰写**：用 STAR 框架写一段 30 秒可说完的「召回低排查」面试回答。

## 验收标准

- 排查 checklist 覆盖 6 个检查项。
- 30 秒面试答法按「现象 → 清单 → 排查点 → 修复」结构组织。
- 可用于模拟面试演练。

## 延伸

准备一个你自己经历过的检索排障故事，写成 Debug Story 文档。
