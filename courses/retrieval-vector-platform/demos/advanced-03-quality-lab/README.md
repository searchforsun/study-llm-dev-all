# Demo：检索质量实验

**章节**：[../../index.html#ch-advanced-03-quality]检索质量实验

## 目标

设计消融实验矩阵，运行 Grid Search，输出标准化质量报告，对比基线结果。

## 实验内容

1. **假设提出**：写一条检索假设（如「ef=128 比 ef=64 提升 recall@10 >= 3%」）。
2. **实验矩阵设计**：创建 `experiment-matrix.csv`，列出 ef=[32,64,128] 和 rewrite=[True,False] 的 6 组组合。
3. **Grid Search 运行**：用 Python 脚本遍历 6 组参数，记录每组 recall@10 和 p95_ms。
4. **质量报告**：用模板生成包含背景、矩阵、结果（表格）、结论和建议的报告。
5. **结论撰写**：读取 `ablation-table.md` 的实验数据，写一段 200 字以内的结论。

## 验收标准

- 实验矩阵至少覆盖 2 个参数变量、每个至少 2 个值。
- 质量报告包含 4 个标准部分。
- 结论中包含明确的推荐参数值及理由。

## 延伸

增加 Rerank 是否开启作为第三个变量，扩展实验矩阵。
