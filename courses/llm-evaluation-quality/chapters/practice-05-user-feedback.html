埋点与标注

user feedback event：thumbs、comment、feedback label。

代码示例

json
{"event":"thumbs_down","trace_id":"t-99","question":"…","labels":["wrong_fact"],"user_segment":"legal"}

坏例回流

badcase reflow：T+1 进入 triage，T+3 进 hard case 或 jsonl。

NS1 飞轮
flowchart LR
  FB[feedback] --> TRI[triage]
  TRI --> DS[dataset vN+1]
  DS --> CI[regression]
  CI --> REL[release]

优先级排序

priority queue：影响用户数 × 严重度 × 合规风险。

  • P0：错误政策承诺
  • P1：事实错误高频
  • P2：体验/格式

双栈发布节奏

release train：Python 索引先 Canary，Spring BFF 跟随；closed loop Weekly 评审 ns1 flywheel。
