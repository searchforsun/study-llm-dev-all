集成测试

spring integration test 读 classpath JSONL 或 testcontainer 索引。

代码示例

java
@SpringBootTest
class RagEvalIT {
  @Test
  void faithfulnessOnGoldenSubset() {
    var report = evalService.runSubset("eval/corpassist-s1-v3.jsonl", 20);
    assertThat(report.getFaithfulness()).isGreaterThan(0.88);
  }
}

Advisor 断言
