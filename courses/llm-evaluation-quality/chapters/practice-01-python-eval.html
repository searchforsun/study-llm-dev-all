pytest + RAGAS

pytest eval：把 evaluate 包在 test 里，或单独 job。

代码示例

python
@pytest.mark.eval
def test_s1_faithfulness(eval_runner):
    report = eval_runner.run("corpassist-s1-v3.jsonl")
    assert report["faithfulness"] >= 0.90

报告

eval report 写 json artifact，含 git sha、prompt_version、index_version。

Python eval CI
flowchart LR

  ART --> S3[artifact store]
  S3 --> SPRING[Spring 对比 job]

阈值闸门

threshold block：任一项低于 yaml 阈值则 pytest.fail。

不推荐
  • 只 print warning
推荐
  • exit 1 block merge

代码示例

yaml
thresholds:
  faithfulness_min: 0.90
  hard_case_faithfulness_min: 0.88
