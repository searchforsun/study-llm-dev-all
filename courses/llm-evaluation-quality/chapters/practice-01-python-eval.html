pytest + RAGAS：评测用例标准化

pytest eval 把 evaluate 包在 test 函数中。每个 test 对应一条评测样本或一组聚合指标的断言。

CorpAssist S1 · 发版前回归 Python 轨每周发版前自动跑 200 条回归集。pytest 收集 eval marker 的测试，依次：① 初始化 eval_runner fixture ② 逐条跑 RAG 管线 ③ 调用 RAGAS 计算指标 ④ 聚合后与阈值比较 ⑤ 生成 eval report 写入 json artifact

# test_s1_eval.py — S1 评测用例
import pytest
import json
from datasets import Dataset

@pytest.mark.eval
class TestS1Regression:
    """S1 回归评测"""

    def test_s1_faithfulness(self, eval_runner, eval_dataset, thresholds):
        """S1 忠实度门禁"""
        dataset = self._build_dataset(eval_dataset)
        scores = eval_runner.evaluate(dataset)
        assert scores["faithfulness"] >= thresholds["faithfulness_min"], \
            f"Faithfulness {scores['faithfulness']:.3f} < {thresholds['faithfulness_min']}"

    def test_s1_context_recall(self, eval_runner, eval_dataset, thresholds):
        """S1 检索召回门禁"""
        dataset = self._build_dataset(eval_dataset)
        scores = eval_runner.evaluate(dataset)
        assert scores["context_recall"] >= thresholds["context_recall_min"], \
            f"Context recall {scores['context_recall']:.3f} < {thresholds['context_recall_min']}"

    def test_s1_hard_case(self, eval_runner, thresholds):
        """难例子集门禁"""
        # 只加载 hard_case.jsonl
        hard_case_dataset = self._load_hard_cases()
        scores = eval_runner.evaluate(hard_case_dataset)
        assert scores["faithfulness"] >= thresholds["hard_case_faithfulness_min"], \
            f"Hard case faithfulness {scores['faithfulness']:.3f} < threshold"

    def _build_dataset(self, samples):
        return Dataset.from_dict({
            "question": [s["question"] for s in samples],
            "answer": [self._mock_answer(s) for s in samples],
            "contexts": [s.get("contexts", []) for s in samples],
        })

    def _mock_answer(self, sample):
        # 实际场景调用 RAG API
        return f"Mock answer for: {sample['question'][:20]}"

    def _load_hard_cases(self):
        hard = []
        with open("./data/hard_cases.jsonl", "r") as f:
            for line in f:
                hard.append(json.loads(line))
        return self._build_dataset(hard)

报告：结构化 JSON Artifact

eval report 写 json artifact，含 git sha、prompt_version、index_version。报告不仅是门禁依据，也是版本基线比较的基础。

代码：评测报告生成

python 
# report_generator.py — 结构化报告生成
import json
import subprocess
from datetime import datetime
from typing import Dict, List

class EvalReport:
    """评测报告生成器"""

    @staticmethod
    def get_git_sha() -> str:
        return subprocess.check_output(
            ["git", "rev-parse", "--short", "HEAD"]
        ).decode().strip()

    @classmethod
    def build(cls, scores: Dict[str, float],
              thresholds: Dict[str, float],
              dataset_version: str,
              passed: bool,
              per_sample: List[Dict] = None) -> Dict:
        return {
            "run_id": f"eval-{datetime.utcnow().strftime('%Y%m%d-%H%M%S')}",
            "git_sha": cls.get_git_sha(),
            "dataset_version": dataset_version,
            "timestamp": datetime.utcnow().isoformat(),
            "aggregate": scores,
            "thresholds": thresholds,
            "passed": passed,
            "sample_count": len(per_sample) if per_sample else 0,
            "per_sample": per_sample or [],
        }

    @staticmethod
    def save(report: Dict, path: str = "./data/eval_report.json"):
        with open(path, "w", encoding="utf-8") as f:
            json.dump(report, f, indent=2, ensure_ascii=False)
        print(f"Report saved: {path}")

    @staticmethod
    def compare(current: Dict, baseline: Dict) -> Dict:
        """对比当前报告与 baseline"""
        deltas = {}
        for metric in current["aggregate"]:
            if metric in baseline["aggregate"]:
                deltas[metric] = round(
                    current["aggregate"][metric] - baseline["aggregate"][metric], 4
                )
        return {
            "deltas": deltas,
            "regression": any(v < 0 for v in deltas.values()),
        }

阈值闸门：阻塞低质量合并

threshold block：任一项低于 yaml 阈值则 pytest.fail，GitHub required check 变红，阻止 MR 合并。

Python 评测流水线
flowchart LR
  PY[pytest eval] --> ART[report.json]
  ART --> GATE{Threshold Gate}
  GATE -->|pass| S3[artifact store]
  GATE -->|fail| FAIL[exit 1 ❌]
  S3 --> SPRING[Spring 对比 job]

pytest eval 作为 GitHub required check，失败时阻塞合并。Spring job 读取同一 artifact 做 cross-stack diff。

阈值 YAML 配置

yaml 
# config/eval-gate.yaml
thresholds:
  faithfulness_min: 0.90
  context_recall_min: 0.85
  answer_relevancy_min: 0.88
  hard_case_faithfulness_min: 0.85

regression:
  baseline_path: ./data/baseline_report.json
  max_delta: -0.02

dataset:
  name: corpassist-s1
  version: v3
  path: ./data/eval_dataset.jsonl

GitHub Actions 配置

yaml 
# .github/workflows/python-eval.yml
name: Python Eval Gate
on: [pull_request]
jobs:
  pytest-eval:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-python@v5
        with:
          python-version: "3.11"
      - run: pip install -r requirements-eval.txt
      - run: pytest tests/test_s1_eval.py -m eval -v --junitxml=report.xml
      - if: always()
        uses: actions/upload-artifact@v4
        with:
          name: eval-report
          path: ./data/eval_report.json
      - if: always()
        uses: actions/upload-artifact@v4
        with:
          name: junit-report
          path: report.xml
不推荐
  • 只 print warning，不阻塞合并
  • 无 baseline 对比，只看绝对分数
  • 每次评测用不同 dataset 版本
  • 无 junit report 不利于 CI 集成
推荐
  • pytest.fail + exit 1 block merge
  • 对比 baseline delta（如 -0.02 以内）
  • dataset_version 与 git tag pin 绑定
  • junit.xml 与 json artifact 双输出

Regression Set 与版本管理

regression set 是 gold 评测集中最稳定的子集。每次发版前跑回归集，确保核心能力不退化。

  • 稳定子集：从完整评测集选 50-100 条最稳定的 QA 对
  • 难例全量：hard_cases.jsonl 全量跑，不允许任何退化
  • 版本锁定：regression set 版本号与 dataset_version 独立，更少更新
  • 基线存储：每次通过的评测报告作为下一次 baseline
regression set 与完整评测集什么关系？

regression set ⊆ 完整集。regression set 用于快速门禁（5 分钟），完整集用于详细报告（15 分钟）。MR 时跑 regression，发版前跑完整集。

本章结论

Python 评测流水线是 NS1 飞轮中「自动评测」的工程实现。pytest + RAGAS + threshold block 形成可重复的 CI 门禁。

  • pytest + RAGAS：@pytest.mark.eval 标记评测用例，eval_runner fixture 封装 RAGAS。
  • 报告：json artifact 含 git sha、dataset_version、per sample 指标。
  • 阈值闸门：pytest.fail -> exit 1 -> GitHub required check 变红，block merge。
  • Regression Set：稳定子集快速门禁 + 难例全量不退化。
