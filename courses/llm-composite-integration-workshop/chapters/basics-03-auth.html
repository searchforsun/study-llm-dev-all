JWT 传递

Gateway 验签；BFF 注入 X-User-Id/Tenant。 关联：jwt forward、service trust。

服务间信任

内网 mTLS 或 signed internal JWT。 关联：tenant isolation、m2m auth。

关键术语

jwt forward · service trust · tenant isolation · m2m auth · user context · internal token · acl header · zero trust

示例

http
POST /v1/rag/query HTTP/1.1
Authorization: Bearer <user-jwt>
X-Tenant-Id: corp-001
X-Request-Id: req-uuid
